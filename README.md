# Grenais Dashboard

Análise dos 25 últimos GreNais (2018-2025) com Python e Streamlit. Os dados foram coletados manualmente, tratados com Pandas e organizados em um dashboard interativo com gráficos e filtros por período.

Fiz esse projeto porque sou gaúcho e queria ter os dados do clássico organizados de um jeito que fizesse sentido pra mim — acabou virando um bom exercício de coleta, limpeza e visualização.

---

## O que o projeto cobre

- Vitórias por time e total de gols por partida
- Artilheiros das partidas
- Histórico cronológico dos jogos

## Stack

Python | Pandas | Streamlit | Matplotlib

## Como rodar

    pip install -r requirements.txt
    streamlit run app.py
